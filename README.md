# 排版工具 — 8种风格一键生成

粘贴任意文本，自动生成8种精美排版，支持6种配色方案，一键导出PNG。

## 在线使用

👉 [点此打开](https://你的用户名.github.io/typograph-tool/)

## 本地使用

1. 下载 `index.html`
2. 浏览器打开即可使用
3. 无需安装任何依赖

## 功能

- **8种排版风格**：经典卡片、时间轴、大卡片、极简留白、杂志风、笔记本、信息图、网格布局
- **6种配色方案**：暖阳粉、星空蓝、森林绿、活力橙、薰衣紫、深海青
- **一键导出PNG**：支持单张或批量导出
- **纯前端**：无需服务器，离线可用（首次加载需网络获取截图库）

## 使用方法

1. 在文本框粘贴你的内容
2. 点击「生成排版」
3. 切换风格和配色
4. 点击「下载当前风格」或「下载全部8种」
